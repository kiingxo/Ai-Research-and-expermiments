## Quick Start Guide

1. **Choose your API** based on your NLP task
2. **Check rate limits** and authentication requirements
3. **Review terms of service** for commercial use
4. **Start collecting data** with provided code examples

## Tips for Dataset Collection

- **Combine multiple sources** for diverse, robust datasets
- **Handle PII carefully** - anonymize personal information
- **Respect rate limits** to avoid IP bans
- **Monitor API costs** - some services charge per request
- **Store raw data** before processing for reproducibility
